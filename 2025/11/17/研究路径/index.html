PlanN 数据分析 log 11.17

Posted on 2025-11-17 In 科研 , R

迂回的14个模型，迷茫的一周与失败的结果。

会赢吗？

摘要

为确定行为抑制性 (BI) 对内化问题（IN）的作用如何受到认知灵活性 (CF) 的调节，进行了如下探索：

数据预处理

  • 针对数据分布偏态：采用 Cox 生存模型残差构建了 BI 连续分数 (BI_score_new_rev)
  • 针对多量表整合出现的数据缺失：利用 MICE 多重插补处理缺失数据。

研究流程

从全样本的连续变量调节模型 (M1-M5) 开始，发现 CF ✖️ BI 交互项虽方向一致但效应微弱且不显著。随后转向基于 BI 聚类分组 (M9) 的简单斜率分析，发现 CF 的保护作用仅在中度BI组中边缘显著 (Estimate = -0.0220, p=0.0543)，但在高 BI 组中效应完全消失 (Estimate ≈ 0.0014)。

探究效应消失原因

进一步构建了针对高 BI 亚组的内部验证模型 (M14)。结果显示，在该子样本中，连续 BI 分数与 CF 的交互作用不显著 (Estimate = -0.0038, p=0.698)。

结论

M14 模型排除了统计功效不足的解释，支持了CF保护机制在行为抑制性达到极端高风险水平时，其功能会停止发挥的理论假设。或许可以解释为 CF 的自发调节作用效应有限，在高 BI 组中更需要进行更早更有效的干预。

研究过程

一、 连续变量调节模型（Model 1 - Model 5）

  • 目的：在全样本中，检验 CF 的效应是否被连续的 BI 分数 (BI_z) 调节。同时，通过逐步增加控制变量，考察核心效应的稳健性。
  • 数据：多重插补后的 mids 对象，所有变量均使用标准化分数或中心化分数。
  • 方法：MICE 后回归，采用 with() 和 pool() 函数。
  • 工作内容：
    • M1：基础模型，检验 age + sex + BI_z ✖️ CF_z。
    • M2：加入家庭情绪环境控制变量（income, SES_subj, edu_parent, dep_parent2）。
    • M3：加入儿童人格/气质控制变量（SPS, extroversion, effort_control, negative_affect, novelty_hesitation）。
    • M4：加入儿童执行功能控制变量（wm_q, inhib_q, ef_total_q, effort_q）。
    • M5：加入情绪调节与外化问题控制变量。

  • 结果：交互项 BI_z ✖️ CF_z 方向为负（支持 CF 保护作用），但在所有模型中均不显著。

二、 聚类调节模型（Model M9）

  • 目的：解决 M1-M5 结果不显著的问题，转而检验 CF 的效应是否仅在特定的 BI 风险亚组中显著（简单斜率检验）。
  • 数据：在 MICE 数据集中加入了 BI 的高斯混合模型聚类结果（BI_cluster，分为高、中、低三组）。
  • 方法：Pool-by-Lapply 方法进行拟合。
  • 工作内容：
    • M9：核心调节模型，检验 IN_log sim Controls + BI_cluster ✖️ CF_centered。以中度 BI 组 (Cluster 1) 为参照组。
  • 结果：
    • 中度 BI 组：CF 的简单斜率边缘显著（Estimate = -0.0220, p=0.0543）。
    • 高 BI 组：CF 的简单斜率趋近于零（Estimate ≈ 0.0014）。
  • 结论：CF 的保护作用是情境依赖的，在高 BI 组中效应缺失。

三、 内部验证模型（Model M14）

  • 目的：解决 M9 结果引发的“高 BI 效应缺失是由于 BI 原始数据信息损失导致的统计功效不足（方差过小），还是CF 机制效应本身过小”这一关键争议。
  • 数据：仅使用 高 BI 组 (Cluster 2) 的子样本，并使用连续的 BI 分数 (BI_score_new_rev) 进行调节分析。
  • 方法：Pool-by-Lapply 方法进行拟合。
  • 工作内容：
    • M14：子样本内部调节模型，检验 IN_log sim Controls + CF_centered ✖️ BI_score_new_rev
  • 结果：
    • 交互项 CF_centered:BI_score_new_rev：Estimate = -0.0038, p = 0.698。（不显著）
  • 结论：由于 CF 效应在高 BI 组内部不被连续 BI 差异调节，彻底排除了“统计功效不足”的解释。

总结

结论：调节效应存在，但形式是非线性的。
